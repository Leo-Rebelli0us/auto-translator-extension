# 自动翻译插件使用教程

## 一、插件简介

自动翻译插件是一个浏览器扩展工具，可以在您浏览网页时快速翻译选中的文本。主要功能包括：

- **自动划词翻译**：选中文本后自动显示翻译结果
- **右键菜单翻译**：通过右键菜单手动触发翻译
- **快捷键操作**：使用快捷键快速翻译
- **翻译弹窗**：显示翻译结果并提供复制、发音等操作

**支持的浏览器**：
- Google Chrome
- Mozilla Firefox  
- Microsoft Edge

**支持的语言**：
- 支持多种语言互译
- 目标语言默认为中文
- 源语言支持自动检测

## 二、安装指南

### Chrome浏览器安装步骤

1. 打开Chrome浏览器，访问扩展页面：`chrome://extensions/`
2. 在页面右上角开启"**开发者模式**"开关
3. 点击"**加载已解压的扩展程序**"按钮
4. 选择插件文件夹（包含`manifest.json`文件的目录）

### Firefox浏览器安装步骤

1. 打开Firefox浏览器，访问附加组件页面：`about:addons`
2. 点击"**扩展**"选项卡
3. 点击齿轮图标 → 选择"**从文件安装附加组件**"
4. 选择插件文件夹

### Edge浏览器安装步骤

1. 打开Edge浏览器，访问扩展页面：`edge://extensions/`
2. 开启"**开发人员模式**"开关
3. 点击"**加载解压缩的扩展**"按钮
4. 选择插件文件夹

### 安装后检查

安装成功后，您应该能在浏览器工具栏看到翻译插件图标。如果看不到，请尝试重启浏览器。

## 三、基本使用

### 1. 自动划词翻译（最常用）

**使用方法**：
1. 在任意网页上用鼠标选中一段文本
2. 等待约0.5秒（防抖处理）
3. 在选中文本下方会自动弹出翻译窗口

**效果**：
- 显示原文和翻译结果
- 支持复制、发音、收藏等操作
- 点击弹窗外任意位置关闭窗口

### 2. 右键菜单翻译

**使用方法**：
1. 在网页上选中文本
2. 右键点击选中的文本
3. 在右键菜单中选择"**翻译选中文本**"
4. 查看翻译结果弹窗

**适用场景**：
- 自动翻译未触发时
- 需要手动控制翻译时机时

### 3. 快捷键操作

**默认快捷键**：
- **Windows/Linux**：`Ctrl + Shift + T`
- **Mac**：`Command + Shift + T`

**使用方法**：
1. 选中网页文本
2. 按下快捷键
3. 查看翻译结果

**注意**：如果快捷键与其他扩展冲突，可以在设置中修改。

### 4. 翻译弹窗功能

翻译弹窗显示时，您可以进行以下操作：

- **复制翻译结果**：点击"复制"按钮，将翻译结果复制到剪贴板
- **语音播放**：点击"发音"按钮，使用浏览器语音合成功能朗读翻译
- **收藏翻译**：点击"收藏"按钮保存翻译记录（功能待完善）
- **关闭弹窗**：点击右上角×按钮或弹窗外任意位置

## 四、配置说明

### 打开设置页面

1. 右键点击浏览器工具栏中的插件图标
2. 选择"**选项**"
3. 或在插件弹窗中点击设置按钮

### 主要配置选项

1. **翻译服务**（默认使用LibreTranslate免费API）
   - API端点：可自定义翻译服务地址
   - API密钥：如需使用付费服务可填写

2. **语言设置**
   - 源语言：自动检测或指定特定语言
   - 目标语言：默认中文，可修改为其他语言

3. **功能开关**
   - 启用自动翻译：开启/关闭自动划词翻译
   - 显示翻译弹窗：控制是否显示翻译窗口
   - 启用右键菜单：控制是否显示翻译右键菜单
   - 启用快捷键：开启/关闭快捷键功能
   - 启用缓存：缓存翻译结果提高速度

### 推荐设置（新手使用）

- 保持所有功能为开启状态
- 使用默认的LibreTranslate API
- 目标语言设置为中文
- 源语言使用"自动检测"

## 五、常见问题

### 1. 翻译不工作怎么办？

**可能原因及解决方法**：
1. **网络问题**：检查网络连接是否正常
2. **API服务问题**：LibreTranslate服务可能暂时不可用，等待一段时间重试
3. **配置错误**：检查设置中的API端点是否正确
4. **浏览器限制**：某些网站可能阻止扩展运行

### 2. 弹窗不显示怎么办？

**检查步骤**：
1. 确认"显示翻译弹窗"选项已开启
2. 尝试重新选中文本，等待足够时间（约0.5秒）
3. 检查网页是否阻止了弹窗显示（某些网站有安全限制）
4. 重启浏览器重试

### 3. 右键菜单不显示怎么办？

**解决方法**：
1. 在设置中确认"启用右键菜单"选项已开启
2. 右键菜单可能需要刷新页面后才生效
3. 尝试重启浏览器

### 4. 快捷键不起作用怎么办？

**排查步骤**：
1. 确认"启用快捷键"选项已开启
2. 检查快捷键是否与其他扩展冲突
3. 尝试修改快捷键配置
4. 重启浏览器

### 5. 如何更换翻译API？

**更换方法**：
1. 打开插件设置页面
2. 在"翻译服务"部分修改API端点
3. 如需API密钥，在相应字段填写
4. 点击"测试连接"验证API是否可用
5. 保存设置

**推荐API**：
- LibreTranslate（免费，默认）
- 其他兼容的翻译API服务
